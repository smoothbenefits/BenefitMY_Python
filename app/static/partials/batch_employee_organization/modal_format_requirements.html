Format Requirements

The batch employee organization import functionality requires exact conformation to the expected input format described below.
The input must follow the below rules:
  1. Each line represents the complete data for organzation setup of one employee
  2. Each line must contain all the required fields below
  3. The fields must be in the order listed out in the table below
  4. The fields must be delimited with a tab
  5. Each line must be ended with the designated "end-record" field
Required Fields:
Field Notes
First Name First name of the account holder
Last Name Last name of the account holder
Manager First Name First name of the manager of the account holder. Note: Leaving Manager Name fields empty indicates the intention of removing the employee's manager assignment, if one exists.
Manager Last Name First name of the manager of the account holder. Note: Leaving Manager Name fields empty indicates the intention of removing the employee's manager assignment, if one exists.
End of Record Annotation Field This is a system required field to clearly indicate the end of a record line. Please simply add a field, "end-record", as the last field of every line.

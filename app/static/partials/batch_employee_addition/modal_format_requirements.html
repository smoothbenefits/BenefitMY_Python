Format Requirements

The batch account creation functionality requires exact conformation to the expected input format described below.
The input must follow the below rules:
  1. Each line represents the complete data for one account
  2. Each line must contain all the required fields below
  3. The fields must be in the order listed out in the table below
  4. The fields must be delimited with a tab
  5. Each line must be ended with the designated "end-record" field
Required Fields:
Field Notes
First Name First name of the account holder
Last Name Last name of the account holder
Email Email of the account holder. Please note that this email will become the user login for the new account
Password Initial password for the new account. If "Send Email" is checked to indicate the desire for the system to automatically send registrition email, then this field must be left blank. Otherwise, it must be specified.
Employment Type The type of employment of the account holder. Must be one of the following values: FullTime and PartTime, case sensitive.
Annual Base Salary The current annual base salary of the account holder. This field must present if and only if the employment type is "FullTime". The format should be a number without the dollar sign.
Hourly Rate The current hourly pay rate for the account holder. This field must present if and only if the employment type is "PartTime". The format should be a number without the dollar sign.
Projected Working Hours per Month The current projected working hours per month for the account holder. This field must present if and only if the employment type is "PartTime". The format should be a number without the dollar sign.
Employment Start Date The date that the employment of the account holder started. Must be in the format of MM/DD/YYYY. E.g. 03/20/2006
Benefit Start Date The date that the benefit for the employee would start take effect. Could be the same as "Employee Start Date". Must be in the format of MM/DD/YYYY. E.g. 03/24/2008
Company Group Must specify the group name the employee belongs to. The group name has to match to the existing group exactly. Otherwise, a new company group will be created with the name
Manager First Name (Optional)First name of the manager of the account holder. Only enter this field (and "Manager Last Name" field) if the account holder has a manager.
Manager Last Name (Optional)First name of the manager of the account holder. Only enter this field (and "Manager First Name" field) if the account holder has a manager.
End of Record Annotation Field This is a system required field to clearly indicate the end of a record line. Please simply add a field, "end-record", as the last field of every line.
